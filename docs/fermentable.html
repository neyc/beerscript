Fermentable.json contains the JSON definition for fermentable ingredients

Definitions

FermentableAddition:

FermentableAddition collects the attributes of each fermentable ingredient for use in a recipe fermentable bill

amount: measureable_units.jsonMassType
add_after_boil: boolean
FermentableBase
GrainOrAdjunct

FermentableBase: object

FermentableBase provides unique properties to identify individual records of fermentable ingredients

name: string

Name of the fermentable item. e.g. 'name':'Vienna malt'

type: string , x ∈ { adjunct , dry extract , extract , grain , sugar , honey }

Type of fermentable. eg. 'type':'grain'

color: measureable_units.jsonColorType

The color of the fermentable. e.g. 'color':{'unit':'EBC','value':4} or 'color':{'unit':'L','value':8}

ibu_gal_per_lb: number

For hopped extracts only - an estimate of the number of IBUs per pound of extract in a gallon of water.

recommend_mash: boolean

true if it is recommended the grain be mashed, false if it can be steeped. A value of true is only appropriate for a 'grain' or 'adjunct' types.

potential: measureable_units.jsonDensityType

The potential yield of the fermentable ingredient. For example “1.036” or “1.040” might be valid potentials.

origin: string

The origin of the fermentable e.g. 'origin':'Belgium'

supplier: string

The supplier of the fermentable e.g. 'supplier':'Castle Malting'
