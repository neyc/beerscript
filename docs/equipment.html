EquipmentBase: object

The descriptive base type for brew day equipment. Provides unique properties to fully describe the recipe.

name: string

The name of your equipment. e.g. Experimental Fermenter 10L

form: string , x ∈ { HLT , Mash Tun , Lauter Tun , Brew Kettle , Fermenter , Aging Vessel , Packaging Vessel , Other }

To specify your equipment, use this form enum as it is specific

type: string

In case the property form doesn't fit set it to Other and use this type property
