Definitions

Recipe: object

Recipe composes the information stored in a beerjson recipe

name: string
type: string , x ∈ { extract , partial mash , all grain }
author: string
coauthor: string
created: measureable_units.jsonDateType
batch_size: measureable_units.jsonVolumeType
boil_size: measureable_units.jsonVolumeType
boil_time: measureable_units.jsonTimeType
efficiency: object
conversion: measureable_units.jsonPercentType
lauter: measureable_units.jsonPercentType
mash: measureable_units.jsonPercentType
brewhouse: measureable_units.jsonPercentType
style: style.jsonRecipeStyleType
ingredients: object
fermentable_bill: object[]

fermentable_bill collects all the fermentable ingredients for use in a recipe

fermentable.jsonFermentableAddition
hop_bill: object[]

hop_bill collects all the hops for use in a recipe

hop.jsonHopAddition
miscellaneous_additions: object[]

miscellaneous_additions collects all the miscellaneous items for use in a recipe

misc.jsonMiscellaneousAddition
culture_additions: object[]

culture_additions collects all the culture items for use in a recipe

culture.jsonCultureAddition
water_additions: object[]

water_additions collects all the water items for use in a recipe

water.jsonWaterAddition
mash: mash.jsonMashProcedure
notes: string
original_gravity: measureable_units.jsonDensityType
final_gravity: measureable_units.jsonDensityType
alcohol_by_volume: measureable_units.jsonPercentType
ibu_estimate: hop.jsonIBUEstimateType
color_estimate: measureable_units.jsonColorType
beer_pH: measureable_units.jsonAcidityType
carbonation: number
fermentation: fermentation.jsonFermentationProcedure
taste: object
notes: string
rating: number
calories_per_pint: number
