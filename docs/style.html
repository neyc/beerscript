RecipeStyleType:

RecipeStyleType defines style information stored in a recipe record

    StyleBase

StyleBase: object

The descriptive base type for both style guideline records, and recipe style provisions. Provides unique properties to identify individual styles

name: string
category: string
category_number: integer
style_letter: string (1 to 1 chars) , must match [A-Z ]
style_guide: string
type: StyleCategories

StyleCategories: string , x ∈ { beer , cider , kombucha , mead , other , soda , wine }
